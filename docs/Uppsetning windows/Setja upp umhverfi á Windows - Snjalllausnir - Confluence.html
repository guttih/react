    Setja upp umhverfi á Windows

    Skip to end of metadata
    • Created by Guðjón Hólm Sigurðsson, last modified just a moment ago
    Go to start of metadata

    Góð síða sem hægt er að leita til: https://facebook.github.io/react-native/docs/getting-started.html

    Nauðsynleg umhverfi og framework

    Atom

    Til að not Visual Studio Code þá þarf að adda plugin fyrir react-native og ESLint plugin:

    Android StudioTil þess að búa til keyranlegt app og keyra app í simulator fyrir Android
    Python 2React-native notar Python 2
    JDKJava Developer Kit, til þess að keyra Android Studio.
    Nodejs / NPMTil þess að keyra JS / Pakki til að sækja og stjórna dependencies.
    WatchmanFylgist með skrám á disk og bíður eftir breytingur til að uppfæra.

    Uppsetning á react-native

    1. Setja upp python2

    Hakaðu við "Add python.exe to Path" í uppsetningarálfi python 2.

    Smelltu hér til sækja og setja upp Python 2.  

    2. Setja upp nodejs 

    Þegar þú ert að setja upp nodejs þá þarf að setja NPM pakkahöndlarann líka.

    Smelltu hér til að sækja og setja upp nodejs.

    Til þess að flýta fyrir þegar þú ert að nota react-native, þá er gott að setja upp yarn.  

      -Uppsetninarálf fyrir yarn má finna hér.  Smellir bara á "Download Installer", hleður honum niður og keyrir.

    3. Setja upp React-native

    Gott er að setja upp yarn, það er ekki nauðsynlegt en flýtir fyrir þegar þú notar react-native.

    Opnaðu Command Promt í Administrative mode (sjá hér) og sláðu inn eftirfarandi skipanir.

    Command Prompt (Administrator)
    ?
    npm update
    npm install -g react-native-cli
     
    ::Optionally install yarn to speedup react-native
    npm install -g yarn

    4. Setja upp Android Studio

    Smelltu hér, sæktu og keyrðu uppsetningarálfinn fyrir Android Studio.

    Veldu Custom installation þegar þú setur upp Android Studio í fyrsta skipti og vertu viss um að haka við eftirfarandi möguleika:

    • Android SDK
    • Android SDK Platform
    • Performance (Intel ® HAXM)
    • Android Virtual Device

    Smelltu svo á "Next" til að setja allt þetta upp.

    5. Setja upp Android 6.0 (Marshmallow) SDK

    Android Studio setur upp nýjasta SDK umhverfið.  React Native notar hinsvegar Android 6.0 (Marshmallow) SDK og þurfum við því að setja það upp.

    a. Opnaðu SDK Manager og hakaðu við að setja upp Android 6.0 (Marshmallow) settu umhverfið upp.

      Hægt er að opna SDK Manager úr "Welcome to Android Studio" skjánum með því að smella á "Configure" → SDK Manager neðst á skjánum.

      Einnig er hægt að opna SDK Manager í Android Studio frá "Tools" flettivalmyndinni, undir Android → SDK Manager.

    b. Veldu flipann "SDK Platforms" í SDK Manager, og hakaðu við "Show Package Details". Finndu og sprengdu út "Android 6.0 (Marshmallow)" færsluna, og sjáðu til þess að hakað sé við eftirfarandi undir atriði:

    • Google APIs
    • Intel x86 Atom System Image
    • Intel x86 Atom_64 System Image
    • Google APIs Intel x86 Atom_64 System Image

    c. Veldu svo flipann "SDK Tools" og hakaðu við "Show Package Details" hér líka. Finndu og sprendu út "Android SDK Build Tools", og sjáðu til þess að hakað sé við "Android SDK Build-Tools 23.0.1".

    d. Afritaðu slóðina í "Android SDK Location" því þú þarft að nota hana  í næsta skrefi.

    e. Að lokum smelltu á "Apply" til að sækja og setja upp Android SDK og tengd build tools.

    6. Setja upp ANDROID_HOME og JAVA_HOME umhverfisbreytunar (environment variable).

    Smelltu á Windows takkann og skrifaðu "environ",

    þá ætti "Edit system environment variables" flýtivalmöguleikinn að birtast.  

    Smelltu á hann.

    Smelltu á takkann "Environment Variables..."

    Smelltu á "New..." undir User variables for..."

    í Variable name skal vera ANDROID_HOME og í Variable Value  á að vera slóðin sem þú aftritaðir áðan í skrefi 5c.

    Smelltu á "New..." undir User variables for..."

    í Variable name skal vera JAVA_HOME og í Variable Value  á að vera slóðin að java JDK, dæmi: "C:\Java\jdk1.8.0_91"

    Gott er að sjá til þess að í PATH breytunni sé slóð á möppuna "platform-tools"   C:\Users\yourUserName\AppData\Local\Android\sdk\platform-tools"

    5. Prófum uppsetninguna þá

    a. Keyrðu Android sýndartæki

    Opnaðu Android AVD Manager í Android Studio frá "Tools" flettivalmyndinni, undir Android → AVD Manager.

    Veldu þér tæki og ýttu á Play takkann til að ræsa það.

    Einnig er hægt að keyra emulator úr command prompt:

    Command Prompt
    ?
    ::Lista emulatora sem til eru
    %ANDROID_HOME%\TOOLS\EMULATOR -list-avds
    ::Dæmi um hverig má keyra emulated device sem heitir "Android_Accelerated_x86"
    start %ANDROID_HOME%\TOOLS\EMULATOR @Android_Accelerated_x86

    Í emulator þá er nauðsynlegt að þekkja þessar keyboard-shortcuts:

     

    • R+R (ýta tvisvar á r til að endurhlaða appinu þínu)
    • Ctrl+M til að fá upp device menu, þar sem Dev Settings er einn möguleikinn


    b. Búa til prufu project

    Opnaðu Command Promt og sláðu inn eftirfarandi skipanir.

    Command Prompt
    ?
    react-native init PrufuProject
    cd PrufuProject
    npm install --save-dev eslint eslint-config-react eslint-plugin-react babel-eslint

    Visual Studio Code

    Til að nota Visual Studio Code þá ætti að sækja og setja upp plugin fyrir react-native og plugin ESLint.

    1. Opna VS-Code
    2. Ctrl+p
    3. Líma þennan texta "ext install vscode-react-native" og smella á "install" takkann við "React Native Tools"
    4. Sláðu í extensions leitargluggann "ESLint" og smelltu á Install við "ESLint"
    5. Smella á "Reload" til að endurræsa VSCode.
    6. Í Windows FileExplorer hægri-smella á "PrufuProject" möppuna og velja open with Code
    7. Skráin index.android.js inniheldur textann sem þú átt að sjá á sýndartækinu þegar þú keyrir projectið.

    d. búa til configuration skrá fyrir ESLint.

    Ýmist er hægt að gefa skipunina "eslint --init" og svara spurningum eða nota búa til skránna ".eslintrc.json" í rót "PrufuProject" og setja þetta efni inn í þá skrá.

    e. Að lokum keyra projectið "PrufuProject"

    Laga allar villur (t.d. breyta spaces í tab) og keyra eftirfarandi skipun.

    Command Prompt
    ?
    react-native run-android

    Það tekur nokkkurn tíma að ræsa projectið, svo haltu hestunum þínum .

     

    Þegar það er komið í gang ætti sýndartækið að sýna þetta.

    Debugging a react-native project

    Þegar búið er að starta Anroid Emulator

    Visual Studio Code

    Ctrl+Shift+D velja "Debug Android" og smella á Play. Hef nú ekki séð hvar hægt er að setja breakpointa, þetta launchar bara appinu.

    1. Starta emulator: sjá: Keyrðu Android sýndartæki

    Í emulator þá er nauðsynlegt að þekkja þessar keyboard-shortcuts:

        • R+R (ýta tvisvar á r til að endurhlaða appinu þínu)

        • Ctrl+M til að fá upp device menu, þar sem Dev Settings er einn möguleikinn

    2. Í windows Explorer Hægri-smella á möppuna sem hefur react-native projectið þitt og velja "Open with Visual Studio Code"
    3. Í  VSCode: Ctrl+Shift+P og skrifa ">React Native: Run Android" 
    4. Á sýndartæki:  "Færð upp rauðan glugga (Unable to connect to remote debugger"
    5. Í VSCode: Ctrl+SHIFT+D, velja "Attach to packager" og smella á Play takkann.
    6. Á sýndartæki: "RR", ýta tvisvar á r til að endurhlaða.
    7. Nú ættir þú að vera komin í debug-mode.

    Quick Reference.  Nýtt project & start Debug session

    Í þessum lið er gert ráð fyrir að búið er að setja  upp react og Visual studio code.  

    Hér er þá bara verið að búa til nýtt project sem heitir project1 og er það keyrt upp.

    Afrita þarf efni úr skránni .eslintrc.json þegar beðið er um það hér fyrir neðan.

    1.  Opnaðu Command Promt og sláðu inn eftirfarandi skipanir.

    Command Prompt
    ?
    ::Lista emulatora sem til eru
    ::   %ANDROID_HOME%\TOOLS\EMULATOR -list-avds
    ::Dæmi um hverig má keyra emulated device sem heitir "Android_Accelerated_x86"
    ::   start %ANDROID_HOME%\TOOLS\EMULATOR @Android_Accelerated_x86
     
    react-native init project1
    cd project1
    echo "//Settu efnið í lið 3 her" > .eslintrc.json
    ::  Afritaðu ofannefnt efni úr skránni .eslintrc.json í skránna sem notepad opnar
    notepad .eslintrc.json
     
    npm install --save invariant warning
    npm install --save-dev eslint eslint-config-react eslint-plugin-react
    npm install babel-eslint && code .
    :: Keyra projectið
    react-native run-android

    2. Ef Visual studio code opnast ekki sjálfkrafa þá farðu í windows Explorer Hægri-smella á möppuna project1 og velja "Open with Code"

    3. Afritaðu þetta efni í skránna  "project1\.eslintrc.json".

    4. Visual Studio  Code:  Ctrl+Shift+D, smella á tannhjólið nálægt Play takkanum og velja "React Native"

    4. Í  Visual Studio  Code: Ctrl+Shift+P og skrifa ">React Native: Run Android" 

      Á sýndartæki:  "Færð upp rauðan glugga (Unable to connect to remote debugger"

    5. Í  Visual Studio  Code: Ctrl+SHIFT+D, velja "Attach to packager" og smella á Play takkann.

    6. Á sýndartæki: "RR", ýta tvisvar á r til að endurhlaða.

    7. nú ættir þú að vera komin í debug-mode.
